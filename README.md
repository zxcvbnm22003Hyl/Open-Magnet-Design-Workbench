# Rat-Workbench
Rat Workbench is developed based on the project RAT. Project RAT is an open-source magnet design project. We have developed an open-source front-end GUI interface called Rat Workbench using Python, enabling magnet design engineers, students and other personnel to quickly conduct magnet design and simulation.
